# JonOS

A portable Linux bootstrap toolkit created by Jon Baldassari.

## Goals

- Rapid deployment of Linux workstations
- Consistent configuration across distributions
- Version-controlled dotfiles
- Automated package installation
- Repeatable system setup

## Supported Platforms

- Arch Linux
- Garuda Linux
- Rocky Linux
- Ubuntu
- Fedora (planned)

## Features

- Fish shell configuration
- Vim/Neovim configuration
- Fastfetch customization
- Git configuration
- Helper scripts
- Bootstrap installer
