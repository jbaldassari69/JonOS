# JonOS

> A portable Linux workstation bootstrap and configuration framework.

JonOS is a personal open-source project focused on creating a consistent, repeatable Linux environment across multiple distributions and platforms.

The goal is to automate workstation deployment while keeping every configuration understandable, maintainable, and fully documented.

---

## Philosophy

JonOS follows a few simple principles:

- One change at a time
- Readable automation
- Clear documentation
- Portable where practical
- Safe defaults
- Version-controlled configuration

---
